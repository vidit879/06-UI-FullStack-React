-> Steps to installl bootstrap
------------------------------
-> https://getbootstrap.com/ -> click on download link -> Compiled CSS and JS -> click on download button
-> extract the downloaded zip file -> copy the "css" , "js" folders and paste in your workspace.
-> <link rel="stylesheet" href="bootstrap/css/bootstrap.css">
-> <script src="bootstrap/js/bootstrap.bundle.js"></script>

-> Steps to installl bootstrap Icons
------------------------------
-> https://getbootstrap.com/ -> click on icons -> click install link -> click on download latest zip -> click on sourceCode.zip -> extract it -> copy the "font" folder and paste it in workspace
-> connect to bootstrap icons
   -> <link rel="stylesheet" href="bootstrap/icons/font/bootstrap-icons.css">
   
   
->  Button -> .btn -> apply styles  

-> Button -> use the class name
